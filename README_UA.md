# Docker with AUTOMATIC1111 (Українська версія)

Цей проєкт є безкоштовним open-source програмним забезпеченням.
Він призначений для автоматизованого розгортання Stable Diffusion WebUI в Docker-середовищі.

Фінансова підтримка проєкту є добровільною та спрямована на компенсацію часу і зусиль автора
за виконану роботу. Жодні товари або ліцензії не продаються.

Вся технічна документація доступна англійською мовою: [English documentation](README.md).
